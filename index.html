B1python by awacha

Python macros for (A)SAXS evaluation

This is a more-or-less coherent package of various functions/objects aimed at processing (anomalous) small-angle x-ray scattering measurements. B1 in the package name hints at beamline B1 at HASYLAB/DESY, Hamburg, Germany. The functions included are loosely based on Ulla Vainio's Matlab scripts for the same purpose, with added features which I (A. Wacha) use during my daily work: fitting, summarizing, extracting, working with different file formats, modeling... I publish it with the hope that it would be useful for other scientists as well, however I cannot (and won't :-) guarantee them to be error- or bug-free.

Dependencies

python 2.x (2.6 works, 2.5 maybe) matplotlib scipy cython enthought/mayavi2 python-ifeffit (optional) pil (Python Imaging Library, optional)

Install

distutils, so the commands python setup.py build and python setup.py install should do the trick. If you are on windows, you can also download the windows installer from PyPI (pypi.python.org)
