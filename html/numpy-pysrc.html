  3  ===== 
  4

  6    1. An array object of arbitrary homogeneous items

 10  How to use the documentation 
 11  ---------------------------- 
 12  Documentation is available in two forms: docstrings provided 
 13  with the code, and a loose standing reference guide, available from 
 14  `the NumPy homepage <http://www.scipy.org>`_. 
 15   
 16  We recommend exploring the docstrings using 
 17  `IPython <http://ipython.scipy.org>`_, an advanced Python shell with 
 18  TAB-completion and introspection capabilities.  See below for further 
 19  instructions. 
 20   
 21  The docstring examples assume that `numpy` has been imported as `np`:: 
 22   
 23    >>> import numpy as np 
 24   
 25  Code snippets are indicated by three greater-than signs:: 
 26   
 27    >>> x = x + 1 
 28   
 29  Use the built-in ``help`` function to view a function's docstring:: 
 30   
 31    >>> help(np.sort) 
 32   
 33  For some objects, ``np.info(obj)`` may provide additional help.  This is 
 34  particularly true if you see the line "Help on ufunc object:" at the top 
 35  of the help() page.  Ufuncs are implemented in C, not Python, for speed. 
 36  The native Python help() does not know how to view their help, but our 
 37  np.info() function does. 
 38   
 39  To search for documents containing a keyword, do:: 
 40   
 41    >>> np.lookfor('keyword') 
 42   
 43  General-purpose documents like a glossary and help on the basic concepts 
 44  of numpy are available under the ``doc`` sub-module:: 
 45   
 46    >>> from numpy import doc 
 47    >>> help(doc)

 82  Viewing documentation using IPython 
 83  ----------------------------------- 
 84  Start IPython with the NumPy profile (``ipython -p numpy``), which will 
 85  import `numpy` under the alias `np`.  Then, use the ``cpaste`` command to 
 86  paste examples into the shell.  To see which functions are available in 
 87  `numpy`, type ``np.<TAB>`` (where ``<TAB>`` refers to the TAB key), or use 
 88  ``np.*cos*?<ENTER>`` (where ``<ENTER>`` refers to the ENTER key) to narrow 
 89  down the list.  To view the docstring for a function, use 
 90  ``np.cos?<ENTER>`` (to view the docstring) and ``np.cos??<ENTER>`` (to view 
 91  the source code). 
 92   
 93  Copies vs. in-place operation 
 94  ----------------------------- 
 95  Most of the functions in `numpy` return a copy of the array argument 
 96  (e.g., `sort`).  In-place versions of these functions are often 
 97  available as array methods, i.e. ``x = np.array([1,2,3]); x.sort()``. 
 98  Exceptions to this rule are documented.
